在idea中使用Maven

2021-03-12

注意！

千万别用idea2020.1版本，坑爹的一批！！！

还有，maven别放在只有管理员才能读写的文件夹下！！

正式开始

1.下载maven

官网下载

官网下载的是压缩包，解压之后放在一个合适的文件中就行；

2.配置环境变量

  • 系统变量:MAVEN_HOME = {maven所在的文件的路径}
  • 系统变量:path = %MAVEN_HOME%\bin

3.检验是否配置成功

然后win+R 运行cmd 输入 mvn -version，成功的话会提示你安装的maven的版本信息和一些其他的信息。

4.配置maven的settings.xml文件

4.1添加本地仓库

首先，在maven文件目录下新建一个repository文件夹

然后找到conf文件夹下的settings.xml文件

<!-- localRepository
   | The path to the local repository maven will use to store artifacts.
   |
   | Default: ${user.home}/.m2/repository
  <localRepository>/path/to/local/repo</localRepository>
  -->

<!--找到上面这些代码，在下面添加如下的代码，表示仓库的位置-->
  <localRepository>D:\Program Files (x86)\apache-maven-3.6.3\repository</localRepository>

4.2添加镜像

<mirrors>
    <!-- mirror
     | Specifies a repository mirror site to use instead of a given repository. The repository that
     | this mirror serves has an ID that matches the mirrorOf element of this mirror. IDs are used
     | for inheritance and direct lookup purposes, and must be unique across the set of mirrors.
     |
    <mirror>
      <id>mirrorId</id>
      <mirrorOf>repositoryId</mirrorOf>
      <name>Human Readable Name for this Mirror.</name>
      <url>http://my.repository.com/repo/path</url>
    </mirror>
     -->
    
    <!--找到上面这些代码，在下面添加如下的代码，表示使用阿里的镜像，提高下载速度-->
	 <mirror>
       <id>nexus-aliyun</id>
       <mirrorOf>*</mirrorOf>
       <name>Nexus aliyun</name>
       <url>http://maven.aliyun.com/nexus/content/groups/public</url>
     </mirror>
  </mirrors>

4.3检验仓库和镜像是否配置成功

win+R 运行cmd 输入mvn help:system测试，成功的话应该会提示下载成功，而且速度很快，可以看到repository文件夹下多了一些东西

5.在idea中配置maven

  • 快捷键 Ctrl+Alt+S，打开设计界面

  • 找到 Build，Execution，Deployment这一栏

  • 再选中Build Tools----Maven，修改即可

千万别用idea2020.1版本！！！！！
